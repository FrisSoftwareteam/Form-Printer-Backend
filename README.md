# form-fill-backend

Minimal Express API used by the Form Printer frontend.

## Setup

```bash
cd "$env:USERPROFILE/Documents/form-fill-backend"
npm install
npm run dev
```

- Dev: http://localhost:4000/health
- Sample: http://localhost:4000/api/hello

## Scripts
- `npm run dev` — start with nodemon
- `npm start` — start with node

Create a `.env` file for secrets if needed.
